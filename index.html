⚔️ Web Sword Cursor

移动鼠标，体验剑形指针！剑会跟随你的移动方向旋转。

当前方向: idle

使用方法

在你的项目中导入并使用：

import { initSwordCursor } from 'web-sword-cursor';

// 1. 使用默认剑
initSwordCursor();

// 2. 从文件加载 SVG
fetch('/path/to/sword.svg')
.then(r => r.text())
.then(svg => initSwordCursor({ swordItem: svg }));

// 3. 传入自定义 SVG 字符串
const svg = `<svg viewBox="0 0 100 100">...</svg>`;
initSwordCursor({ swordItem: svg });
